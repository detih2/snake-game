## Детали по каждому файлу

### 1. vc-ru-article.md

Полноценная статья для публикации на VC.ru:

- **Формат:** ~1000 слов, неформальный тон
- **Структура:**
  - Проблема (скучные промокоды)
  - Решение (геймификация через змейку)
  - Механика работы
  - Психология (IKEA-эффект, дофамин, ностальгия)
  - Первые результаты (конверсия x3-5)
  - Примеры использования
  - Тарифы
  - CTA
- **Особенности:**
  - Ссылка на демо в начале
  - Самоирония
  - Конкретные цифры
  - Призыв к действию в конце

### 2. telegram-posts.md

5 готовых постов для разных каналов:

1. **Для маркетологов** — фокус на growth hacking, метрики
2. **Для разработчиков** — side-project, технический стек
3. **Для бизнеса** — кейс, ROI, цены
4. **Вирусный пост** — короткий, развлекательный
5. **Анонс запуска** — для своего канала

Включена таблица с рекомендациями по времени публикации.

### 3. product-hunt.md

Полный набор материалов для запуска на Product Hunt:

- **Tagline:** 3 варианта (60 символов)
- **Description:** 260 символов
- **First Comment:** история создания, механика, CTA
- **Topics:** 8 релевантных топиков
- **Шаблоны ответов** на частые вопросы
- **Чеклист** медиа-материалов
- **Launch checklist**

### 4. social-media-kit.md

Тексты для всех платформ:

- **Twitter/X:** 3 варианта (280 символов)
- **LinkedIn:** пост 300-500 символов
- **Instagram:** сторис + пост с хештегами
- **Email subject lines:** 10 вариантов
- **Хештеги:** русские и английские
- **Короткие описания** для био/профиля
- **Цитаты** для использования
